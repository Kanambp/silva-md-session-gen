HOW IT WORKS

Four Steps to Go Live

Simple, fast, and secure. Your WhatsApp bot session generated and delivered right to your phone.

01
Choose Method
Pairing Code or QR — both produce the same session.
02
Authenticate
Link your WhatsApp using the code or by scanning the QR.
03
Receive Session
Session ID lands in your WhatsApp DM — compressed and ready.
04
Deploy Bot
Paste as an env variable on any platform and go live.

Choose Your Path

RECOMMENDED

Pairing Code

Enter your phone number, get an 8-digit code, and link in under 10 seconds. No phone camera needed.

1
Enter phone with country code
2
Copy the 8-digit pairing code
3
Open WhatsApp → Linked Devices
4
Session ID sent to your DM
